Etiquetas de Control (formularios y Entradas)
Etiqueta Descripción Propiedades Ejemplo
<form> Contenedor para controles que envían datos al servidor. action, method (GET/POST), enctype.
<input type="text"> Campo de texto de una sola línea. type, name, placeholder, value, required, maxlength.
<input type="email"> Campo para capturar emails (valida formato). type="email", name, required, placeholder.
<textarea> Campo de texto multilínea. rows, cols, placeholder, maxlength.
<select> <option> Lista desplegable y sus opciones. name, multiple, size; option value, selected.
<button> Botón interactivo (submit, button, reset). type, onclick, disabled.
<label> Etiqueta textual asociada a un control de formulario (accesibilidad). for (id del control).
